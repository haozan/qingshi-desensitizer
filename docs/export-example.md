# 导出功能示例

此文件展示了导出 MD 功能生成的 Markdown 文件格式。

## 导出的 Markdown 文件示例

当用户点击"导出MD"按钮时，系统会生成以下格式的 Markdown 文件：

```markdown
# 脱敏文本

> 导出时间: 2026/02/14 01:23:45  
> 工具: 脱敏大师  
> 规则数量: 3 个

---

## 文本内容

【机构A】代理【人物A】诉讼案件，涉及【机构B】商业合同纠纷...

---

## 脱敏规则

1. **北京天元律师事务所** → `【机构A】`
2. **张三** → `【人物A】`
3. **上海XX公司** → `【机构B】`

---

*本文件由 [脱敏大师](https://qinglion.ai) 生成*
```

## 功能特性

1. **文件格式**: 标准 Markdown (.md) 格式
2. **文件命名**: 自动使用时间戳命名，如 `脱敏文本_1708012345678.md`
3. **元数据信息**: 包含导出时间、工具名称、规则数量
4. **内容结构**: 
   - 脱敏后的文本内容
   - 完整的脱敏规则映射表
   - 工具来源链接
5. **字符转义**: 自动转义 Markdown 特殊字符，确保显示正确

## 使用场景

- 将脱敏文本和规则保存为文件备份
- 分享给团队成员查看
- 作为项目文档的一部分归档
- 导入到其他支持 Markdown 的工具中
