# 脱敏大师 - 产品特性与价值总结

## 产品概述

**脱敏大师**是一款专为律师、咨询顾问等专业人士设计的文本脱敏与还原工具。它能够帮助用户安全地将包含敏感信息的文档交给AI工具处理，并在处理后完美还原原始信息，让专业工作更高效、更安全。

---

## 核心价值主张

### 1. **数据安全保护**
在AI时代，专业人士需要借助AI工具提升工作效率，但直接将包含客户姓名、机构名称、地址等敏感信息的文档交给AI处理存在巨大的隐私和合规风险。脱敏大师通过智能脱敏技术，让您能够：
- ✅ 安心使用ChatGPT、Claude等AI工具处理专业文档
- ✅ 完全消除敏感信息泄露风险
- ✅ 满足律师执业规范和数据保护法规要求

### 2. **完美的工作流闭环**
不仅仅是脱敏，更重要的是能够**完美还原**。我们提供双向处理能力：
- **脱敏阶段**：将"张三"替换为"【人物A】"，将"北京天元律师事务所"替换为"【机构A】"
- **还原阶段**：AI处理后，将文本中的"【人物A】"还原为"张三"，"【机构A】"还原为"北京天元律师事务所"
- **映射记录**：所有脱敏映射关系自动保存，支持跨会话使用

### 3. **零学习成本，极致体验**
- 🖱️ **所见即所得**：鼠标选中敏感文字，系统自动高亮全文所有相同内容
- ⚡ **一键操作**：批量添加规则后，点击"一键脱敏"即可完成，点击"一键还原"即可恢复
- 💾 **自动保存**：脱敏规则本地持久化存储，关闭页面后依然保留
- 🎯 **智能匹配**：支持长短文本混合匹配，自动按长度优先处理，避免冲突

## 核心功能特性

### 一、智能文本选择与高亮

**功能描述**：
- 在原文区域用鼠标选中任意敏感文字
- 系统自动统计并显示全文中该文字出现的次数（如"找到 3 处"）
- 实时反馈，帮助用户快速识别需要脱敏的内容

**技术实现**：
- 使用Stimulus框架的`text_highlight_controller`
- 基于浏览器原生Selection API
- 正则表达式精确匹配全文

**用户价值**：
- 节省手动查找时间
- 避免遗漏敏感信息
- 直观的视觉反馈

---

### 二、批量脱敏规则管理

**功能描述**：
- 为每个选中的敏感文本设置自定义脱敏标签（如【机构A】、【人物B】、【地址C】）
- 支持一次性添加多个脱敏规则
- 规则列表清晰展示"原文 → 标签"的映射关系
- 支持单个删除或一键清空所有规则

**标签格式规范**：
- 强制使用**中文方括号【】**格式
- 原因：AI工具（如ChatGPT、Claude）能更好地识别和保留这种标签格式
- 系统会自动验证标签格式的合法性

**技术实现**：
- 使用`desensitization_manager_controller`管理规则状态
- 基于localStorage实现本地持久化存储
- 防重复添加逻辑，自动检测已存在的规则

**用户价值**：
- 批量处理效率高
- 规则可复用，无需重复设置
- 即使关闭浏览器，规则依然保留

---

### 三、一键脱敏处理

**功能描述**：
- 添加完所有规则后，点击"一键脱敏"按钮
- 系统自动将原文中的所有敏感信息替换为对应的脱敏标签
- 脱敏结果实时显示在右侧区域
- 支持一键复制脱敏文本到剪贴板

**智能替换算法**：
- 按文本长度降序处理，避免短文本误匹配长文本
- 全局替换，确保所有出现位置都被处理
- 正则转义，支持特殊字符

**技术实现**：
```typescript
// 按长度降序排序，防止重叠匹配
const sorted = [...this.mappingRules].sort((a, b) => 
  b.original_text.length - a.original_text.length
)

sorted.forEach(rule => {
  const regex = new RegExp(this.escapeRegex(rule.original_text), 'g')
  result = result.replace(regex, rule.desensitized_label)
})
```

**用户价值**：
- 秒级完成大批量文本脱敏
- 精确替换，无遗漏
- 可安全地将脱敏文本交给AI工具处理

---

### 四、一键完美还原

**功能描述**：
- AI处理后的文本（包含脱敏标签）粘贴到右侧区域
- 点击"一键还原"按钮
- 系统自动将所有脱敏标签还原为原始敏感信息
- 还原结果实时显示在左侧原文区域

**智能还原算法**：
- 按标签长度降序处理，避免误匹配
- 支持AI对文本的任意编辑、重组、扩展
- 只要标签格式保持不变，即可精确还原

**使用场景示例**：
```
脱敏前：
北京天元律师事务所代理张三诉讼案件，涉及上海XX公司商业合同纠纷。

脱敏后（交给AI）：
【机构A】代理【人物A】诉讼案件，涉及【机构B】商业合同纠纷。

AI处理后（可能重新组织句子）：
本案由【机构A】作为【人物A】的代理方，针对【机构B】提起商业合同纠纷诉讼。

还原后：
本案由北京天元律师事务所作为张三的代理方，针对上海XX公司提起商业合同纠纷诉讼。
```

**用户价值**：
- AI处理文本再长也能瞬间还原
- 不限制AI对文本的编辑和改写
- 确保最终交付物包含真实信息

### 五、100%离线本地存储

**安全设计理念**：
- **零服务器交互**：所有脱敏规则和映射记录保存在浏览器localStorage
- **永不上传**：敏感数据从不离开您的设备
- **离线可用**：无需联网即可使用所有功能（PWA支持）

**数据持久化**：
```typescript
// 存储键名
private readonly STORAGE_KEY = 'desensitization_mappings'

// 保存规则
private saveMappings(): void {
  localStorage.setItem(this.STORAGE_KEY, JSON.stringify(this.mappingRules))
}

// 加载规则
private loadMappings(): void {
  const stored = localStorage.getItem(this.STORAGE_KEY)
  if (stored) {
    this.mappingRules = JSON.parse(stored)
  }
}
```

**用户价值**：
- 绝对的数据安全保障
- 满足律师执业保密义务
- 符合GDPR、PIPL等数据保护法规

### 六、Progressive Web App (PWA) 能力

**安装与离线功能**：
- ✅ 支持"添加到主屏幕"，像原生应用一样使用
- ✅ Service Worker离线缓存，无网络也能访问
- ✅ 桌面和移动端均可安装（Windows、Mac、iOS、Android）
- ✅ 独立窗口运行，无浏览器地址栏干扰

**PWA特性**：
- **快速启动**：预缓存核心资源（CSS、JS、图标）
- **离线访问**：缓存关键页面（首页、脱敏工具页）
- **自动更新**：Service Worker版本管理，自动清理旧缓存
- **推送通知**：支持Web Push Notifications（基础设施已就绪）

**安装方式**：
- 桌面端：点击导航栏"安装应用"按钮
- 移动端（Android）：点击"安装应用"按钮
- 移动端（iOS）：Safari → 分享 → 添加到主屏幕

**技术实现**：
- Manifest配置：中文名称、主题色、图标、快捷方式
- Service Worker策略：
  - HTML页面：Network-First（优先网络，失败回退缓存）
  - 静态资源：Cache-First（优先缓存，7天过期）
- 安装按钮：自动检测已安装状态，智能显隐

**用户价值**：
- 随时随地可用，不依赖网络
- 应用级体验，更专业
- 数据完全本地化，更安全

### 4. **可扩展性**
- 模块化架构：控制器职责清晰，易于维护
- Playbook支持：通过playbook快速扩展新功能

## 数据安全与合规

### 隐私保护
- ✅ **零数据上传**：所有数据处理在客户端完成
- ✅ **本地存储**：使用浏览器localStorage，数据不跨域
- ✅ **无服务器日志**：服务器端不记录任何用户输入
- ✅ **无第三方追踪**：不使用Google Analytics等追踪工具

### 法规合规
- ✅ **GDPR（欧盟通用数据保护条例）**：数据主体完全控制
- ✅ **PIPL（中国个人信息保护法）**：本地处理，无跨境传输
- ✅ **律师执业规范**：满足律师保密义务要求
- ✅ **医疗隐私法规**：适用于医疗健康场景

### 安全最佳实践
- 🔒 **HTTPS强制**：PWA要求HTTPS，确保传输加密
- 🔒 **CSP策略**：内容安全策略防止XSS攻击
- 🔒 **无Cookie**：不使用Cookie追踪用户
- 🔒 **开源透明**：代码可审计，无后门
